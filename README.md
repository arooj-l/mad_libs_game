# Mad Libs Game (CLI)

A fun, interactive Python command-line game that lets users create silly stories by filling in blanks with their own words.

---

## Features
- User provides inputs for **name, place, animal, adjective**  
- Story template dynamically filled  
- Uses Python **string methods**:
  - `.upper()` → makes NAME dramatic  
  - `.lower()` → makes animal and adjective lowercase  
  - `.title()` → capitalizes the first letter of PLACE  
- Interactive and fun CLI experience

---

## How to Run
1. Make sure **Python 3.x** is installed  
2. Navigate to the project folder in terminal  
3. Run the program:

```bash
python main.py
